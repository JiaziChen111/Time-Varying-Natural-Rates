VAR functions

rvar

Estimate new reduced-form VAR object.

Syntax

[w,y] = rvar(d,list,range,...)
[w,y] = rvar(x,range,...)

Output arguments

w[reduced-form VAR]New reduced-form VAR object.
y[tseries]Multivariate time series with input data and estimated reduced-form residuals.

Input arguments

d[struct]Database with input time series.
list[char | cellstr]List of time series names from the database.
x[tseries]Time series whose columns will be used as input data.
range[numeric]Date range, i.e. IRIS serial date numbers.

Optional input arguments

'cointeg'[numeric | empty]Impose cointegrating vectors (in rows) on the estimated VAR.
'constant'[true | false]Include vector of constants.
'constraints'[char | empty]Linear constraints on the VAR's parameters.
'covparameters'[true | false]Compute covariance matrix for parameter estimates.
'dummyobs'[numeric | empty]Priors imposed as dummy observations.
'elist'[char | cellstr | empty]Give names to residuals.
'maxiter'[numeric | 10]Maximum number of iterations for GLS.
'mean'[numeric | empty]Impose a mean on the estimated VAR.
'order'[numeric | 1]Order of VAR.
'tolerance'[numeric | 1e-5]Termination tolerance on parameters for GLS.
'ylist'[char | cellstr | empty]Give names to endogenous variables.

Description

  • The names given to the VAR's endougenous varibles and residuals through the 'ylist' and 'elist' options are used by the fprintf and sprintf functions.

VAR functions
svar
Identify structural VAR object based on reduced-form VAR object.
Syntax
[w,y] = svar(w,y,...)
Output arguments
w[structural VAR]Structural VAR object.
y[tseries]Multivariate time series with input data and estimated structural residuals.
Input arguments
w[reduced-form VAR]Structural VAR object.
y[tseries]Multivariate time series with input data and estimated reduced-form residuals.
Optional input arguments 
'method'['chol' | 'qr']Use Cholesky or QR decomposition to identify structural shocks.
'ordering'[numeric | 1:n]Ordering of VAR variables.
'std'[numeric | 1]Size of std dev for structural shocks.
