Tseries functions

convert

Convert time series from one frequency to another.

Syntax

xx,newfreq
xx,newfreq,range,...

Output arguments

x[tseries]Time series based on a new frequency.

Input arguments

x[tseries]Time series to be converted.
newfreq[numeric | 'y' | 'z' | 'q' | 'm']Requested frequency.
range[numeric]Date range, i.e. IRIS serial date numbers.

Optional input arguments for hig to low frequency conversion

'method'[@mean | function_handle]Function to be used for aggregating high-frequency data.

Optional input arguments for low to high frequency conversion

'method'['cubic' | 'linear' | 'nearest' | 'spline']One of the methods of Matlab's INTERP1 function to be used for interpolating low-frequency data.

Description

Tseries functions
hpf
Conditional Hodrick-Prescott filter.
Syntax
[tnd,gap] = hpf(x)
[tnd,gap] = hpf(x,range,...)
[gap,tnd] = hpf2(x)
[gap,tnd] = hpf2(x,range,...)
Output arguments
tnd[tseries]Estimated low-frequency component.
gap[tseries]Estimated high-frequency component.
Input arguments
x[tseries]Time series to be filtered.
range[numeric | Inf]Date range, i.e. IRIS serial date numbers.
Optional input arguments 
'growth'[tseries | empty]Pre-sample, in-sample, or post-sample hard tunes imposed on first difference of trend.
'lambda'[numeric | 100*get(x,'freq')^2]Smoothing parameter (noise to signal variance ratio).
'level'[tseries | empty]Pre-sample, in-sample, or post-sample hard tunes imposed on level of trend.
'log'[true | false]Filter applied to the logarithm of the data.
Description
The hpf2 version of the filter swaps the two output arguments, otherwise it is identical to hpf.

llf
Conditional local-level filter.
Syntax
[tnd,gap] = llf(x)
[tnd,gap] = llf(x,range,...)
[gap,tnd] = llf2(x)
[gap,tnd] = llf2(x,range,...)
Output arguments
tnd[tseries]Estimated low-frequency component.
gap[tseries]Estimated high-frequency component.
Input arguments
x[tseries]Time series to be filtered.
range[numeric | Inf]Date range, i.e. IRIS serial date numbers.
Optional input arguments 
'drift'[numeric | 0]Drift (deterministic growth).
'growth'[tseries | empty]Pre-sample, in-sample, or post-sample hard tunes imposed on first difference of trend.
'lambda'[numeric | 10*get(x,'freq')]Smoothing parameter (noise to signal variance ratio).
'level'[tseries | empty]Pre-sample, in-sample, or post-sample hard tunes imposed on level of trend.
'log'[true | false]Filter applied to the logarithm of the data.
Description
The llf2 version of the filter swaps the two output arguments, otherwise it is identical to llf.

userdata
Get or set user data attached to a tseries object.
Syntax
x = userdata(this)
this = userdata(this,x)
Output arguments
x[any]User data currently contained in the tseries object.
this[tseries]Tseries object with user data updated.
Input arguments
this[tseries]Tseries object you want to get or set user data for.
Description
You can attach any kind of user data to tseries objects, and retrieve or change them at any later time.
Example
In the example below, we assume that x is an existing tseries object.
>> d = struct();
>> d.created = datestr(now());
>> d.source = 'National Statistics Bureau, Revision A2.';
>> x = userdata(x,d);
>> userdata(x)
ans = 
    created: '21-Oct-2008 03:05:31'
     source: 'National Statistics Bureau, Revision A2.'

x12
Census X12-ARIMA seasonal adjustment or filter.
Syntax
[x,output,error] = x12(x)
[x,output,error] = x12(x,range,...)
Output arguments
x[tseries]X12-ARIMA filtered series.
output[cellstr]Contents of X12 output file(s).
error[cellstr]Contents of X12 error file(s).
Input arguments
x[numeric]Data organised columnwise.
range[numeric | Inf]Date range, i.e. IRIS serial date numbers.
Optional input arguments 
'arima'[numeric | 0]Time series are to be extended with this number of periods of forecasts and backcast using estimated ARIMA.
'delete'[true | false]Delete X12 input and output files.
'display'[true | false]Display X12 messages in command window.
'mode'['add' | 'mult' | 'pseudoadd' | 'logadd' | 'auto']Seasonal factor model: additive, multiplicative, pseudo-additive, log-additive, or automatic selection.
'output'['d10' | 'd11' | 'd12' | 'd13']Requested output.
'tdays'[true | false]Control for trading-day effect.
